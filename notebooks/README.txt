En esta sección se encuentran los jupyter notebooks utilizados para el desarrollo de los modelos:

- Preprocesamiento: Este notebbok contiene todas la operaciones de preprocesamiento de los datos
- EDA: Este notebook contiene el Analisis exploratorio de datos a partir de la data preprocesada en el paso anterior
- Modelo_LSTM: Este notebook contiene el desarrollo del modelo LSTM para la predicción de demanda de energía
- Modelo_Estadisitico: Este notebook contiene el desarrollo de los modelos estadisticos para la predicción de demanda de energía
- Modelo_Random_forest: Este notebook contiende el desarrollo del modelo de Random Forest para la predicción de demanda de energía
